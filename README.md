### 🌟 Why Git?
* **Performance:** Optimized for speed, handling massive projects (like the Linux Kernel) with ease.
* **Security:** Uses SHA-1 hashing to ensure the history of your code is immutable.
* **Flexibility:** Allows for sophisticated branching and merging strategies.

## 🐧 How to Run on Linux

Follow these steps to execute the scripts on a Linux system (Ubuntu recommended):

### 🔹 Step 1: Navigate to project folder

```bash
cd oss-audit-git
```

### 🔹 Step 2: Give execution permission

```bash
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
```

### 🔹 Step 3: Run the scripts

```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/apt/history.log git
./script5.sh
```

---

### ⚠️ Notes

* Ensure you are using a Linux environment (WSL or native Ubuntu)
* Some scripts may require access to system log files
* Use `sudo` if permission errors occur
